## 如何运行

1. 请确认已经安装好python，并有如下科学运算包```pandas```、```seaborn```，其中```pandas```需要依赖```xlrd```以读取Excel文件，依赖```openpyxl```以写入Excel文件
2. 如果您不能确保以上项目包均以安装，可以在项目Terminal里直接运行```pip install -r requirements.txt```安装依赖包
